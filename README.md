# Laundry Services Responsive Issues

A responsive Laundry Services landing page built using HTML and CSS. This project focuses on creating a modern, user-friendly, and fully responsive hero section that adapts seamlessly across desktop, tablet, and mobile devices.

## 🚀 Features

- Responsive Navigation Bar
- Modern Hero Section Design
- Call-to-Action Button
- Mobile-Friendly Layout
- Flexbox-Based Structure
- Media Queries for Different Screen Sizes
- Responsive Image Handling
- Clean and Simple User Interface

## 🛠️ Technologies Used

- HTML5

- Media Queries

## 📱 Responsive Design

This project includes responsive layouts for:

### Desktop View
- Full navigation menu
- Side-by-side hero section layout

### Tablet View (≤ 768px)
- Reduced font sizes
- Optimized spacing
- Responsive image scaling

### Mobile View (≤ 480px)
- Hidden navigation links
- Stacked hero section layout
- Full-width button
- Mobile-optimized typography
